# Projeto de Intervenção – PEC Sem Duplicidade

Trabalho desenvolvido pelos internos de Medicina da UNIVASF na UBS Amaro Ivaldo, como projeto de intervenção do rodízio de Medicina de Família e Comunidade, sob a orientação dos residentes e do coordenador da unidade.

## Objetivo
Contribuir para a identificação e correção de duplicidades nos cadastros do Prontuário Eletrônico do Cidadão (PEC), com o objetivo de aprimorar a organização das informações, garantir a continuidade do cuidado e aumentar a eficiência dos serviços de saúde prestados.
